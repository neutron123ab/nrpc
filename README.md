# nRPC —— 自制rpc框架
技术栈
* 网络通信：netty
* 注册中心：nacos
* 序列化：protostuff

## 协议：

| magic_code | version | length       | messageType | codec      | compress     | requestId | body         |
| ---------- | ------- | ------------ | ----------- | ---------- | ------------ | --------- | ------------ |
| 魔数       | 版本号  | 报文头部长度 | 消息类型    | 序列化方式 | 数据压缩方式 | 请求id    | 消息内容     |
| 4字节      | 1字节   | 4字节        | 1字节       | 1字节      | 1字节        | 4字节     | 根据实际情况 |
